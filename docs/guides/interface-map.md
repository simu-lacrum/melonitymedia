# Interface Map — Карта экранов

## Обзор зон

| Зона | URL-префикс | Доступ | Описание |
|------|-------------|--------|----------|
| Публичная | `/` | Все | Landing page |
| Авторизация | `/auth/*` | Неавторизованные | Вход / Регистрация |
| Пользователь | `/account/*` | JWT (role: USER \| ADMIN) | Панель вебмастера |
| Администратор | `/admin/*` | JWT (role: ADMIN) | Управление платформой |

---

## Публичная зона

| Роут | Компонент | Описание |
|------|-----------|----------|
| `/` | `page.tsx` | Landing page: hero + возможности + инфраструктура + CTA |

---

## Авторизация (`/auth`)

| Роут | Компонент | Описание |
|------|-----------|----------|
| `/auth/login` | `auth/login/page.tsx` | Форма входа (email + пароль). JWT → HttpOnly Cookie |
| `/auth/register` | `auth/register/page.tsx` | Форма регистрации. Редирект → `/account/dashboard` |

---

## Панель вебмастера (`/account`)

### Dashboard (`/account/dashboard`)
**Назначение:** «Взгляд сверху» на всю сетку.

| Блок | Контент |
|------|---------|
| KPI Cards (×4) | Просмотры, Живые аккаунты, Подписчики, Видео |
| Recharts график | Динамика просмотров / подписчиков за 30 дней |
| Последние задачи | Лента недавних BullMQ jobs со статусами |

---

### Профили (`/account/profiles`)
**Назначение:** Единый инвентарь аккаунтов.

| Функция | Описание |
|---------|----------|
| DataTable | Чекбокс, Платформа, Никнейм, Статус, Прокси, Просмотры |
| Импорт (Drawer) | Drag-and-Drop зона для `log:pass` файлов |
| Bulk Actions | Массовая привязка прокси, удаление |
| Empty State | Красивая заглушка «Нет аккаунтов» |

---

### Рабочая область (`/account/workspace`)
**Назначение:** Сердце платформы — настройка и запуск задач.

| Блок | Описание |
|------|----------|
| **Глобальные настройки** | Потоки, задержка старта (от/до), профиль/пресет |
| **Медиатека** | Drag-n-Drop загрузка видео (.mp4) |
| **Вкладка А: Залив** | Пулы названий, описаний, тегов. Лимит в сутки |
| **Вкладка Б: Прогрев** | Хештеги, ползунки вероятности (лайк/коммент 0-100%) |
| **Вкладка В: Cookies** | Ссылки на доноров, время на сайте. Кнопка «Скачать ZIP» |
| **Вкладка Г: Профиль** | Аватар, Баннер, Био. Массовое применение |
| **Терминал** | Live Socket.io логи воркеров |
| **Запуск** | Кнопка «🚀 ЗАПУСТИТЬ ЗАДАЧУ» |

---

### Прокси (`/account/proxies`)
**Назначение:** Управление пулом прокси.

| Функция | Описание |
|---------|----------|
| DataTable | Host, Port, Статус, Тип (статический/ротация), Аккаунты |
| Drawer | Добавление: host:port:login:pass + rotation link |
| Тест | Проверка соединения с прокси |
| Ротация | Ручная смена IP по rotation link |

---

## Зона администратора (`/admin`)

### Runtime (`/admin/runtime`)
| Метрика | Источник |
|---------|----------|
| PostgreSQL | `SELECT 1` healthcheck |
| Redis | `PING` |
| BullMQ | Active / Waiting / Completed counts |
| CPU / RAM | `os.cpus()` + `process.memoryUsage()` |

### Users (`/admin/users`)
| Функция | Описание |
|---------|----------|
| DataTable | Email, Роль, Аккаунты, max_threads, Статус |
| Лимиты | Изменение `max_threads_limit` |
| Soft-Ban | Мгновенная отмена задач + выброс из сессии |

### Firewall (`/admin/firewall`)
| Функция | Описание |
|---------|----------|
| IP Blacklist | Список заблокированных IP |
| Добавление | Ввод IP → сохранение в Redis |
| Middleware | Express middleware → `403 Forbidden` |
